New WID on UE Conformance - IoT NTN enhancements plus CT1 aspects
3GPP TSG RAN Meeting #103 RP-240784
Source: CMCC, MediaTek, CAICT
項目 内容
Justification
  • Release 17 RAN1/2/3/4 work item LTE_NBIOT_eMTC_NTNでは、NB-IoTとeMTCがNon-Terrestrial Networks (NTN)をサポートできるようにするために実施された。
  • Release 18 RAN1/2/3/4 work item IoT_NTN_enhでは、NB-IoT NTNとeMTC NTNをさらに強化することが定義されている。
  • Release 18 CT1 work item 5GSAT_Ph2では、以下の内容が定義されている:
    1. 不連続カバレッジによるカバレッジ外期間を決定・交渉し、カバレッジギャップ中の省電力化を行うための手順
    2. カバレッジの喪失と多数のUEが同時にカバレッジに戻ることによるシグナリングオーバーロードの処理
  • Rel-18 RAN1/2/3/4 work item IoT_NTN_enhの全体完成度は、RP#102後にコアパートが100%、パフォーマンスパートが10%に達している。パフォーマンスパートの完了目標はRP#104。
  • Rel-18 CT1 work item 5GSAT_Ph2の全体完成度はCT#102後に90%に達しており、CT#103までに完了する見込み。
  • したがって、IoT NTN拡張UE要件の適合性試験をRAN5仕様に導入することが妥当である。
Objective 本RAN5ワークアイテムの目的は、RAN1、RAN2、RAN4、CT1によって導入されたR18 IoT NTN拡張UE要件に対応するIoT NTN拡張UEの適合要件、テストケース、適用性、テスト環境、テストポイントを定義し、関連する適合仕様を更新することである。
eMTC NTNの明確な商用展開への関心が現時点ではないため、eMTC拡張の適合性テスト開発は本WIの範囲外とみなされる。
Expected Output and Time scale Impacted existing TS/TR:
- TS 36.508, 36.521-3, 36.521-4, 36.521-2, 36.509, 36.523-1, 36.523-2, 36.523-3: Introduction of test cases, test applicability, ICS, testing functions, etc. for Rel-18 IoT NTN enh. (Target: TSG RAN#110)
- TR 36.903: Introduction of TT&MU for RRM test cases impacted by Rel-18 IoT NTN enh. (Target: TSG RAN#110)
